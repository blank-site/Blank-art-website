== Description ==

Easy Tagger plugin lets subscribers to your blog add tags, search posts by them, display 
search results in nice image grid (for that one has to previusly set Featured Image
for each post to display), it also shows some basic statistics like most popular tags,
most active users, recent tags, etc.

= Notice =
Plugin creates tables `wp_users_blacklist` and `wp_users_terms_posts` for it own purposes
check if those don`t already exist in your wordpress installation (some other plugin).

== Installation ==

Follow the steps below to install the plugin.

1. Upload the plugin directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Easy Tagger" option panel to configure the plugin
